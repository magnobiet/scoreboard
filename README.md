## TL;DR

### Development

[![Open in CodeSandbox](https://img.shields.io/badge/Open%20in-CodeSandbox-blue?style=flat-square&logo=codesandbox)](https://githubbox.com/advanced-react-course/boilerplate)

```bash
npm run dev
npm run test:watch
npm run storybook
```

### Build

```bash
npm run build:prod
npm run build-storybook
npm run start
```
